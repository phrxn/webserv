--- Regras para o arquivo de configuracao ---

  ** O nosso servidor vai suportar de 1 a N ServerConfig
  ** Criar a classe CheckConfiguration


--- Regras para interface entre o Servlet e o arquivo de configuracao ---
Perguntas:
  1) O caminho eh valido ou nao? (NOT_FOUND 404)
  2) O caminho tem um redirecionamento? 
  3) O caminho aceita o metodo HTTP ...? (NOT_ALLOWED 405)
  4) O caminho eh um CGI?
  5) Pegar o caminho absoluto a partir do caminhoURI.
  6) Pode listar diretorios?

  A) pegar uma pagina de erro para o codigo tal (400, 300);

Interface CheckConfiguration:

  bool oCaminhoEhValid(String caminhoURI);
        // Tem que checar no arquivo de configuracao se o caminhoURI eh valido ou nao.
        // Nesse caso o metodo deve retorna se o caminhoURI eh valido ou nao.
        // Cabe a esse metodo fazer os "matches", e descobrir nos location a validade ou nao do caminho

  string oCaminhoEhUmRedirecionamento(caminho URI);
        // Vai checar se o caminhoURI eh um redirecionamento, se for, esse metodo deve retornar o caminho
        // "verdeiro" para o recurso, exemplo se: /a/b deve ser redirecionado para www.google.com
        // Esse metodo deve retornar www.google.com
        // Se nao houver nenhum redirecionamento, deve retornar um string vazia

  bool oMetodoHTTPEhPermitido(HTTP_METHOD method, String caminhoURI);
        // Deve checar se um o caminho dado, aceita o metodo HTTP ou ano
        // Deve retornar: true se sim, e false se nao

  bool oCaminhoApontaParaUmCGI(String caminhoURI);
        // Deve dizer se o caminho aponta para um CGI, a classe URL pode ajudar nisso

  string obterOCaminhoAbsolutoDoCaminhoURI(String caminhoURI);
        // Como se sabe, a URI nao eh de fato a representacao para o caminho absoluto, logo,
        // esse metodo deve "converter", ou seja, pegar a URI, pesquisar nas location e retornar o caminho absoluto
        // Por exemplo:  /pasta/a.txt, pode dar match com:  /textos/de_matematica/pasta
        // Logo, /pasta/a.txt, na verdade, vai apontar para  /textos/de_matematica/pasta/a.txt
        // E por obvio, se a URI apontar para um caminho fisico mesmo, esse metodo deve retornar a URI
        // Enfim esse metodo deve converter o caminho URI para o caminho verdadeiro

  bool  aListagemDeDiretorioEstaHabilitadaParaEsseCaminho(CaminhoURI);
        // Nesse caso o Chamador deve garantir que o caminhoURL eh um caminho para um diretorio
        // Esse metodo deve retornar se a listagem de diretorios estah habilitada para esse caminho URL


  string getCaminhoParaUmaPaginaHTTPParaUmStatusHTTP(HTTP_STATUS codigoHTTP);
        // Esse metodo vai receber um codigo HTTP e ele deve retornar um caminho para uma pagina HTTP
        // Que possa existir para esse metodo, se nenhum caminho vai estabelecido
        // Ele deve retornar uma string vazia
