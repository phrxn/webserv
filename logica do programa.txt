1) Virtual host:
  - tem que retornar a porta
  - tem que retornar se o caminho é um diretório
  - tem que dizer se o caminho aponta para um CGI  (NOTHING | PHP | PYTHON)
  - tem que retornar se o caminho é um "desvio"
  - tem que dizer se o host pode listar diretorios
  - tem que dizer um verbo HTTP é suportado por ele (GET | POST | DELETE)
  - tem que retornar o host
  - o tamanho máximo do corpo do Request
  - o tamanho máximo de arquivos para UPLOAD


2) O HTTPRequest:
  - Verificar se a sintaxe está correta ou não
      - Verificar se o verbo é valido (escrita dele)  : 400 - BadRequest
      - Verificar se a URL é valido ou não            : 400 - BadRequest
      - Verificar se a Versao é valido ou não         : 400 - BadRequest
      - Se a primeira linha já nao for valida         : 400 - BadRequest
  - Ver se tem a linha do Host
      - Se nao tem a linha do do Host:          : 400 - BadRequest
  - Procura pelo VirtualHost no VirtualHostFactory usando o host
      - Se nao existir:                         : 400 - BadRequest
  - Pegar o ConfigurationSingleton
  - Ver se o servidor implementa o verbo
      - Se nao implementar                      : 501 - Not Implemented
  - Ver se no VirtualHost se o verbo é implementado
      - Se nao implementado:                    : 405 - Method Not Allowed
  - Ver se tem a linha Content-Lenght
      - Se tiver buscar no VirtualHost o tamanho máximo
      - Se ultrapassar o tamanho maximo         : 413 - Request Entity Too Large


3) O arquivo de configuracao:
	Uma porta pode atender de 1 a N hosts
    Se houver mais de um mesmo HOST para uma porta, o que deve ser considera é o primeiro que aparece no arquivo

		if (VirtuaHost.isUrlAPathToCGI(HTTP.getURL()))
			servlet = new HttpServletCGI(VirtualHost);
		else
			servlet = new HttpServletStatic(VirtualHost);

		-- Agora, baseado no metodo HTTPRequest, chamar o metodo no HttpServlet
		switch (HTTPRequest.getMethod())
		{
		case GET:
			servlet.doGet(HTTPRequest &request, HTTPResponse &response);
			break;
		case POST:
			servlet.doPost(HTTPRequest &request, HTTPResponse &response);
			break;
		case DELETE:
			servlet.doDelete(HTTPRequest &request, HTTPResponse &response);
			break;
		default:
			break;
		}

		-- checar se o Status do HTTPReponse eh OK, se nao for, um erro ocorreu, logo, vamos tratar esse erro aqui!
		if ( HTTPResponse.getStatus() != OK)
			doError(CodeError, HTTPResponse response);
	}
